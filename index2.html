คือตัวที่ช่วยจัดรูปแบบและตกแต่งหน้าเว็บของเราบน browser ซึ่งการเขียน css ก็สามารถเขียนได้หลายๆแบบ ไม่ว่าจะเป็นการเขียน โดยใช้ selector หรือจะเป็นการเขียนแบบ inline ซึ่งผลลัพธ์ที่ออกมาก็สามารถใช้งานได้เหมือนกัน

CSS Specificity คืออะไร ?

Css Specificity คือค่าบางอย่างที่จะจะบอก browser ว่าให้เลือกใช้ Style ตัวไหนมาแสดงผลหน้าเว็บ ซึ่งจะมีการเขียนหลายแบบมากๆ ซึ่งแต่ละแบบจะมีค่า specificity ที่แตกต่างกันออกไป




Css Specificity


CSS Specificity เป็นคอนเซปต์การเลือก CSS Selector ที่มี ความสามารถเยอะที่สุด มาใช้กับ Element บนหน้าเว็บไซต์ ภาษา CSS เหมือนภาษาการเขียนโปรแกรมอื่น ๆ ที่อ่านโค้ดจาก บน – ล่าง แปลว่า ถ้าเขียน CSS Selector ซ้ำกัน 2 อัน อันล่างจะถูกนำไปใช้งานเพราะใหม่กว่า แต่ความพิเศษของ CSS คือ Selector ที่แตกต่างกันแต่ละ Selector จะมีความสามารถ ไม่เหมือนกัน และเมื่อไหร่ที่ความสามารถ ต่างกัน ลำดับการเขียนโค้ดจะไม่มีผลทันที

ตัวอย่างง่าย ๆ ของเรื่อง CSS Specificity คือ ถ้าเราเขียน HTML/CSS แบบนี้:

HTML:
<\div class="box" id="redbox">

CSS:
.box { background: white; } /* ให้ .box พื้นหลังสีขาว */
#redbox { background: red; } /* ให้ #redbox พื้นหลังสีแดง */

โดยในตัวอย่างทั้ง 2 นั้น CSS Selector ทั้ง 2 อันขในตัวอย่างจะมีความสามารถแตกต่างกัน โดยความสามารถจะแตกต่างกันจากการใช้ Class, ID, !Important หรือ การซ้อน Element หลายๆชั้น

ตัวอย่างการเรียนใช้ Class selector

.list-unstyled {
padding-left: 0;
list-style: none;
}

ตัวอย่างการเรียกใช้ ID selector

#content {
flex-direction: column;
}




สรุป

CSS Specificity บราวเซอร์จะเลือกใช้คำสั่งที่เจาะจงที่สุดซึ่งวิธีการจะคำนวนจาก ค่า CSS Selector เพื่อนำมาส่งให้บราวเซอร์รู้ว่าจะใช้ Style ตัวไหนเพื่อนำไปแสดงหน้าเว็บไซต์ หากว่าโค๊ดนั้นมีการเรียกใช้ซ้ำกันเยอะจนเกินไป บราเซอร์จะเลือกใช้คำสั่งที่เจาะจงที่สุด
